About News Classifier

How It Works

This application uses a machine learning model trained on the BBC News dataset to classify news articles into one of five categories:

  • Business: Finance, markets, companies, economy
  • Entertainment: Movies, music, celebrities, arts
  • Politics: Government, elections, policies, legislation
  • Sport: Sports events, teams, athletes, matches
  • Tech: Technology, gadgets, software, innovation

Technology Stack

  • Backend: Flask, Python
  • Machine Learning: Scikit-learn, Naive Bayes
  • Text Processing: NLTK, TF-IDF Vectorization
  • Frontend: Bootstrap, JavaScript

Model Performance

The classifier achieves 99.1% accuracy on the test dataset and can handle articles with multiple category relevance.
